# Aman Verma - Portfolio Website

A modern, fast, and responsive personal portfolio built with **Astro**. This website showcases my projects, technical skills, and experience as a web developer.

## 🚀 Live Website

**https://amanverma.online**

## ✨ Features

* Responsive design
* Modern dark UI
* Fast performance
* SEO optimized
* Custom 404, 403, 500 & 503 pages
* Contact section
* Project showcase
* Skills & Tech Stack sections
* Smooth scrolling and animations

## 🛠 Tech Stack

* Astro
* TypeScript
* CSS
* Cloudflare Pages

## 📂 Featured Projects

### GymForge

A collection of useful fitness tools including calculators and workout utilities.

### Attendance Calculator

A simple web tool to calculate attendance percentage and required classes.

## 🖥 Local Development

Clone the repository:

```bash
git clone https://github.com/amanverma1807/portfolio-site.git
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

# Aman Verma - Portfolio Website

![Portfolio Preview](./README-assets/portfolio-preview.png)

A modern, responsive, and performance-focused portfolio built with Astro to showcase my projects, skills, and web development journey.

## 🌐 Live Demo

👉 https://amanverma.online

## ✨ Features

- Modern responsive design
- Fast performance with Astro
- Clean UI & smooth animations
- SEO optimized
- Custom error pages (403, 404, 500, 503)
- Project showcase
- Skills & Tech Stack section
- Contact section

## 🛠️ Tech Stack

- Astro
- TypeScript
- CSS
- Cloudflare Pages

## 🚀 Local Development

```bash
npm install
npm run dev
```
